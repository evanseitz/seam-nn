Main Tutorial
------------

A complete walkthrough of SEAM using the DeepSTARR model to analyze local regulatory mechanisms:

* Data preprocessing and model loading
* Attribution map generation
* Mechanism clustering and visualization
* MSM generation and interpretation
* Advanced visualization techniques

The full tutorial is available in multiple formats:

* `Interactive Google Colab <https://colab.research.google.com/drive/1HlkEdCpnlMFF_Ab7y7HDDVFS6sIv8c9f?usp=sharing>`_
